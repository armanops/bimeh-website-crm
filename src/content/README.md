# Content - MDX Files

Marketing content and Product Detail Pages (PDPs) for insurance products.

## Structure

Each product gets its own MDX file with metadata frontmatter.

## File Format

```mdx
---
title: Product Name
slug: product-slug
description: Brief description
category: category-name
featured: true/false
---

# Content here...
```

## Processing

Files are processed by ContentLayer2 and served as static pages or reusable components.
